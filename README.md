# Interactive Wall Calendar

A beautifully crafted, feature-rich interactive wall calendar component built with Next.js and Tailwind CSS.

Live Demo: [calender.jeetdas.site](https://calender.jeetdas.site)

## Overview

This project is an interactive calendar application that allows users to manage their months with a premium, tactile feel. It supports date range selection, persistent notes for both months and specific date ranges, and a dynamic theme system that changes based on the current month and user preference.

## Features

- Dynamic Month Themes: Unique color palettes and high-quality hero backgrounds for every month.
- Range Selection: Select a start and end date to get an automatic summary of the duration.
- Persistent Notes: Add notes for the entire month or for specific date ranges. Notes are saved locally and persist between sessions.
- Dark Mode: A fully integrated dark mode that adapts the calendar's aesthetics for low-light environments.
- Responsive Design: Optimized for all screen sizes, from mobile devices to large desktops.
- Fluid Animations: Smooth transitions and hover effects for a premium user experience.
- Today Navigation: Quick shortcut to return to the current date and month.

## Tech Stack

- Framework: Next.js 16+ (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Icons: SVG based custom icons
- State Management: React Hooks (useReducer, custom hooks)
- Persistence: Browser LocalStorage

## Getting Started

First, install the dependencies:

```bash
npm install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
